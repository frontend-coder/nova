
Стартовый шаблон для сайтов с лого и горизонтальным меню в header
start-template-3-blocks-scss

Этот шаблон сделан для того, чтобы на его основе собирать адоптивные шаблоны.
Для облегчения работы уже создана разметка header, она сотоит из фиксированного горизонтального блока, в нем размещены:

лого-картинка
горизонтальное меню для больших экранов и мобильных устройств. Для активации мобильного меню предусмотрена кнопка-гамбургер.
кнопка заказа звонка


В качестве фона header используется изобращение width = 1920рх

Верстка с помощью препроцессора scss, сборщик - prepros.

В каталоге scss размещены следующие файлы:

var.scss - размещены переменные и миксины
main.scss - файл в который интегрированы все файлы scss, в него вставлять пользовательские стили оформления разметки не рекомендовано, можно инклудить стили с остальных файлов *.scss
madia.scss - для придания полной адаптивности шаблона
header.scss - стили оформления header шаблона, в нем уже размещены стили по умолчания, их рекомендовано отредактировать под верстаемый макет psd
fonts.scss - выполняется привязка индивидуальных шрифтов
allstyle.scss - прописываются все стили блоков оформления шаблона, которые идут по разметке ниже header 

Если выполняется верстка сложного макета, разделите файл allstyle на несколько дополнительных, не забудьте их привязать в файле main.scss

Как собрать продакшен версию шаблона:

в файле libs.css - вставлены все стили подключенных модулей:

bootstrap сетка 3.3.7 
font awesome 4.6.3
magnific popup 
Owl Carousel v2.1.6

Весь код минимизирован и оптимизирован.

Пользовательские стили собраны в файл main.css. Их необходимо минимизировать и вставить в файл libs.css

Удалите строку из файла index.html, которая привязывает файл main.css

Блок подключения скриптов

К стартовому шаблону привязаны такие script-плагины

jquery-2.1.3.js
owl carousel.js
equalHeights.js
nicescroll.js
magnific popup.js
PageScroll2id.js
common.js

Во время сборки продакшт версии необходимо все содержимое файлов собрать последовательно в один js файл, к примеру, all-common.js, код минимизирован, поэтому это не нужно делать самостоятельно.

При необходимости подключите другие плагины к стартовому шаблону для этого уже созданы файлы libs.css и jquery-2.1.3.js. 

Во время сборки были к стартовому шаблону включены только самые популярные и необходимые компоненты, которые могут потребоваться для верстки простых и средних по сложности Landing Pages
